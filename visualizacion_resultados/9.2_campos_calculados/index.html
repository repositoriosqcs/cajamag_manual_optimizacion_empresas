    Campos Calculados¶

    En esta sección se presentan los campos calculados o columnas creadas en DAX para la construcción de la capa visual.

    1. EstadoDes

    Tabla: Historico

    Descripción: Estado calculado de los registros.

    
IF(Historico[estado] = "A", "Activo", 
   IF(OR(Historico[estado] = "I", Historico[estado] = "M"), "Inactivo", 
      IF(Historico[estado] = "P", "Pendiente", BLANK())))
    2. KeyPromedioAporte

    Tabla: Historico

    Descripción: Llave calculada para promedio de aportes.

    
RIGHT(Historico[cod], LEN(Historico[cod]) - 2) & LEFT(Historico[Tipo], 3)
    3. RangoNumTrabajadores

    Tabla: Historico

    Descripción: Clasificación de empresas según el rango de trabajadores.

    
if [NumTrab] = null then null else 
if [NumTrab] < 11 then "Micro" else 
if (11 <= [NumTrab]) and ([NumTrab] <= 50) then "Pequeña" else 
if (51 <= [NumTrab]) and ([NumTrab] <= 200) then "Mediana" else 
if 200 < [NumTrab] then "Grande" else null
    4. tipcaptra

    Tabla: Historico

    Descripción: Tipo de capacidad transaccional.

    
IF(Historico[captra] = "N", "Normal", 
   IF(Historico[captra] = "I", "Discapacidad", Historico[captra]))
    5. tipparent

    Tabla: Historico

    Descripción: Relación familiar del beneficiario.

    
IF(Historico[parent] = "1", "Hijo", 
   IF(Historico[parent] = "2", "Hermano", 
      IF(Historico[parent] = "3", "Padre")))
    6. c_ANS

    Tabla: HistoricoANS

    Descripción: Clasificación dentro o fuera del ANS según reglas de negocio.

    
IF(
    (HistoricoANS[Fuente] = "subsi15" && 
     SELECTEDVALUE(Historico[calsuc]) <> "INDEPENDIENTE" && 
     SELECTEDVALUE(Historico[calsuc]) <> "PENSIONADO" && 
     SELECTEDVALUE(Historico[calsuc]) <> "FACULTATIVO" && 
     HistoricoANS[DiasTrabajados] > 8) || 
    (HistoricoANS[Fuente] = "subsi23" && 
     HistoricoANS[DiasTrabajados] > 8),
    "Fuera de Ans",
    IF(
        (HistoricoANS[Fuente] = "subsi15" && 
         SELECTEDVALUE(Historico[calsuc]) = "INDEPENDIENTE" && 
         HistoricoANS[DiasTrabajados] > 3) || 
        (HistoricoANS[Fuente] = "subsi15" && 
         SELECTEDVALUE(Historico[calsuc]) = "PENSIONADO" && 
         HistoricoANS[DiasTrabajados] > 3) || 
        (HistoricoANS[Fuente] = "subsi15" && 
         SELECTEDVALUE(Historico[calsuc]) = "FACULTATIVO" && 
         HistoricoANS[DiasTrabajados] > 3) || 
        (HistoricoANS[Fuente] = "subsi02" && 
         HistoricoANS[DiasTrabajados] > 3),
        "Fuera de Ans",
        "Dentro de Ans"
    )
)
    7. DiasTrabajados

    Tabla: HistoricoANS

    Descripción: Total de días hábiles trabajados considerando festivos.

    
NETWORKDAYS(HistoricoANS[fecafi], HistoricoANS[fecapr], 1, FestivosColombia)
    8. NoAfil

    Tabla: HistóricoAportesEmp

    Descripción: Identificación de empresas no afiliadas que realizaron aportes.

    
IF(LOOKUPVALUE(Historico[KeyMoraEmp], Historico[KeyMoraEmp], HistoricoAportesEmp[KeyHistoricos]) = BLANK(), 1, 0)
    13. DiasMora

    Tabla: HistoricoMoraEmp

    Descripción: Cálculo de días de mora según periodo.

    
VAR nit = HistoricoMoraEmp[nit]
VAR periodo = HistoricoMoraEmp[periodo_actualizacion]
VAR fech_Comp = EOMONTH(EDATE(TODAY(), -2), 0)
VAR PerAnt = CALCULATE(
    MAX(HistoricoMoraEmp[periodo_actualizacion]),
    ALL(HistoricoMoraEmp),
    HistoricoMoraEmp[Primero] = 1,
    periodo >= HistoricoMoraEmp[periodo_actualizacion],
    nit = HistoricoMoraEmp[nit]
)
RETURN DATEDIFF(
    DATEVALUE("01/" & RIGHT(PerAnt, 2) & "/" & LEFT(PerAnt, 4)),
    fech_Comp,
    DAY
)
    14. EdadesPresuntas

    Tabla: HistoricoMoraEmp

    Descripción: Clasificación de mora en rangos según días.

    
IF(
    AND(HistoricoMoraEmp[DiasMora] >= 30, HistoricoMoraEmp[DiasMora] <= 60), "30-60 días",
    IF(
        AND(HistoricoMoraEmp[DiasMora] > 60, HistoricoMoraEmp[DiasMora] <= 90), "61-90 días",
        IF(HistoricoMoraEmp[DiasMora] > 90, "> 91 días", BLANK())
    )
)
    15. NoConvenio

    Tabla: HistoricoMoraEmp

    Descripción: Identificación de empresas sin convenio registrado.

    
IF(LOOKUPVALUE(HistoricoRecuperado[KeyXML4], HistoricoRecuperado[KeyXML4], HistoricoMoraEmp[KeyConvenio]) = BLANK(), 1, 0)
    21. numGiro

    Tabla: MetasCuota

    Descripción: Clasificación de giros por mes usando orden ascendente y densidad.

    
RANKX(FILTER('MetasCuota', 'MetasCuota'[Mes] = EARLIER('MetasCuota'[Mes])), 'MetasCuota'[fecha], , ASC, Dense)
    22. Ano-Cuatri

    Tabla: Periodos

    Descripción: Concatenación del año con el cuatrimestre.

    
Periodos[Año] & "-" & Periodos[Cuatrimestre]
    23. Ano-Trim

    Tabla: Periodos

    Descripción: Concatenación del año con el trimestre.

    
Periodos[Año] & "-" & Periodos[Trimestre]
    24. Cuatrimestre

    Tabla: Periodos

    Descripción: Clasificación del mes dentro de su cuatrimestre correspondiente.

    
VAR mes = VALUE(RIGHT(Periodos[periodo], 2))
VAR cctrim = IF(
    AND(1 <= mes, mes <= 4), 1,
    IF(
        AND(5 <= mes, mes <= 8), 2,
        IF(AND(9 <= mes, mes <= 12), 3, BLANK())
    )
)
RETURN cctrim
    25. Trimestre

    Tabla: Periodos

    Descripción: Clasificación del mes dentro de su trimestre correspondiente.

    
VAR mes = VALUE(RIGHT(Periodos[periodo], 2))
VAR cctrim = IF(
    AND(1 <= mes, mes <= 3), 1,
    IF(
        AND(4 <= mes, mes <= 6), 2,
        IF(
            AND(7 <= mes, mes <= 9), 3,
            IF(AND(10 <= mes, mes <= 12), 4, BLANK())
        )
    )
)
RETURN cctrim
    26. Polaridad

    Tabla: Satisfaccion

    Descripción: Clasificación de la polaridad de las calificaciones.

    
IF([Calificacion] >= 4, "Satisfecho", "Insatisfecho")
