    Indicadores y Medidas Calculadas¶

    A continuación, se presentan los indicadores y medidas calculadas, junto con su descripción, fórmula y meta (si aplica).

    1. _%desafiliaciones

    Descripción: Participación porcentual de desafiliación por motivo.

    
SUM(Historico[seRetiro]) / CALCULATE(SUM(Historico[seRetiro]), REMOVEFILTERS(EstadosDim[detest]))
    2. _%ErroresAfiliaciones

    Descripción: Porcentaje de errores de grabación encontrados en relación con la muestra validada.

    
_%ErroresAfiliacione = SUM('ErroresGrabación'[NumErrores]) / [_muestra]
    3. _%ParDesafiliacion

    Descripción: Participación de la desafiliación en relación con el total de las afiliaciones.

    
CALCULATE(DISTINCTCOUNT(Historico[cod]), Historico[seRetiro] = 1) / DISTINCTCOUNT(Historico[cod])
    4. _incMesActualvsAnteriorAfilMesAnterior

    Descripción: Comparación del resultado entre el periodo de estudio y el periodo anterior para los individuos activos en el mes anterior.

    
IFERROR(
    VAR periodoActual = MAX(Periodos[periodo])
    VAR periodoAnterior = CALCULATE(MAX(Periodos[periodo]), ALL(Periodos), Periodos[periodo] < periodoActual)
    VAR afilAnt = CALCULATE([_totalAfiliados] - CALCULATE(SUM(Historico[seAfilio])), ALL(Periodos), Periodos[periodo] = periodoAnterior)
    RETURN ([_totalAfiliados] - CALCULATE(SUM(Historico[seAfilio])) - afilAnt) / afilAnt,
    "-"
)
    5. _incMesActualvsAnteriorAfilMesAnterior(Año)

    Descripción: Comparación del resultado entre el periodo de estudio y el mismo periodo año anterior para los individuos activos en el mes anterior.

    
IFERROR(
    VAR formula = (
        VAR periodoActual = MAX(Periodos[periodo])
        VAR periodoAnoAnterior = (LEFT(periodoActual, 4) - 1) * 100 + RIGHT(periodoActual, 2)
        VAR afilAnt = CALCULATE([_totalAfiliados] - CALCULATE(SUM(Historico[seAfilio])), ALL(Periodos), VALUE(Periodos[periodo]) = periodoAnoAnterior)
        RETURN IF(ISBLANK(afilAnt), BLANK(), ([_totalAfiliados] - CALCULATE(SUM(Historico[seAfilio])) - afilAnt) / afilAnt)
    )
    RETURN IF(ISBLANK(formula), "-", formula),
    "-"
)
    6. _incMesActualvsAnteriorAfilNuevas

    Descripción: Comparación del resultado entre el periodo de estudio y el periodo anterior para los individuos que se afiliaron en el periodo de estudio.

    
IFERROR(
    VAR periodoActual = MAX(Periodos[periodo])
    VAR periodoAnterior = CALCULATE(MAX(Periodos[periodo]), ALL(Periodos), Periodos[periodo] < periodoActual)
    VAR afilAnt = CALCULATE(SUM(Historico[seAfilio]), ALL(Periodos), Periodos[periodo] = periodoAnterior)
    RETURN (SUM(Historico[seAfilio]) - afilAnt) / afilAnt,
    "-"
)

    10. _totalAfiliados

    Descripción: Número de individuos que continúan o estuvieron afiliados en el mes.

    
DISTINCTCOUNT(Historico[cod])
    11. _indicadorAfilOportunasEmp

    Descripción: Total de afiliaciones de empresas gestionadas dentro del ANS.

    
CALCULATE(
    COUNTROWS(HistoricoANS),
    FILTER(HistoricoANS, HistoricoANS[c_ANS] = "Dentro de Ans" && HistoricoANS[Tipo] = "Empresa"),
    REMOVEFILTERS(Historico[Tipo])
) / 
CALCULATE(
    COUNTROWS(HistoricoANS),
    FILTER(HistoricoANS, HistoricoANS[Tipo] = "Empresa"),
    REMOVEFILTERS(Historico[Tipo])
)
    12. _indicadorNoAfilOportunasEmp

    Descripción: Porcentaje de no cumplimiento afiliaciones oportunas empresas.

    
1 - [_indicadorAfilOportunasEmp]
    13. _#EmpNoAfiliadas

    Descripción: Empresas no afiliadas que aportaron en el periodo de estudio.

    
VAR per = SELECTEDVALUE(Periodos[Periodo])
RETURN CALCULATE(
    DISTINCTCOUNT(HistoricoAportesEmp[nit]),
    ALL(Periodos),
    HistoricoAportesEmp[periodo] = per,
    HistoricoAportesEmp[NoAfil] = 1,
    REMOVEFILTERS(Historico[validador], Historico[Tipo])
)
    14. _#EmpNoAfiliadas(Año)

    Descripción: Empresas no afiliadas que aportaron en el año.

    
VAR per = SELECTEDVALUE(Periodos[Año])
RETURN CALCULATE(
    DISTINCTCOUNT(HistoricoAportesEmp[nit]),
    ALL(Periodos),
    HistoricoAportesEmp[Año] = per,
    HistoricoAportesEmp[NoAfil] = 1
)
    15. _$EmpNoAfiliadas

    Descripción: Total de aportes recibidos en el mes de empresas no afiliadas.

    
VAR per = SELECTEDVALUE(Periodos[periodo])
RETURN CALCULATE(
    SUM(HistoricoAportesEmp[valorpag]),
    ALL(Periodos),
    REMOVEFILTERS(Historico[validador]),
    HistoricoAportesEmp[periodo] = per,
    HistoricoAportesEmp[NoAfil] = 1
)
    16. _$EmpSiAfiliadas

    Descripción: Total de aportes recibidos en el mes de empresas afiliadas.

    
SUM(HistoricoAportesEmp[valorpag])
    17. _aportesActualvsAnterior(Año)

    Descripción: Relación entre el total de aportes recibidos en el año con respecto al año anterior.

    
VAR anoAn = VALUE(MAX(Periodos[Año])) - 1
VAR devAn = CALCULATE(
    SUM(HistoricoAportesEmp[valorpag]) + [_$EmpNoAfiliadas(Año)],
    ALL(Periodos),
    VALUE(Periodos[Año]) = anoAn
)
VAR devAc = SUM(HistoricoAportesEmp[valorpag]) + [_$EmpNoAfiliadas(Año)]
RETURN IF(ISBLANK(devAn), BLANK(), (devAc - devAn) / devAn)

    28. _indicadorAportesActualvsAnterior

    Descripción: Relación entre el total de aportes recibidos en el mes con respecto al mes anterior.

    Meta: Todos los tableros

    
VAR Anio = VALUE(MAX(Periodos[Año]))
VAR mes = VALUE(RIGHT(MAX(Periodos[Periodo]), 2))
VAR perAn = IF(mes = 1, (Anio - 1) * 100 + 12, Anio * 100 + (mes - 1))
VAR devAn = CALCULATE([_$EmpSiAfiliadas] + [_$EmpNoAfiliadas], ALL(Periodos), VALUE(Periodos[periodo]) = perAn)
VAR devAc = [_$EmpSiAfiliadas] + [_$EmpNoAfiliadas]
RETURN IF(ISBLANK(devAn), BLANK(), (devAc - devAn) / devAn)

    40. _indicadorCarteraRecuperada(Año)

    Descripción: Monto presuntamente recuperado en relación con el total de mora presunta en el año para esa cartera.

    Meta: 4

    
[_Recuperado_presunto(Año)] / HistoricoMoraEmp[_Mora_presunta(Año)]
    41. _Mora_presunta(Año)

    Descripción: Mora presunta en el año para esa cartera.

    Meta: Integrado

    
CALCULATE(SUM(HistoricoMoraEmp[valormor]), HistoricoMoraEmp[Ultimo] = 1)
    42. _Recuperado

    Descripción: Dinero pagado de empresas en convenio.

    Meta: Ver ficha técnica de indicadores de madurez

    
SUM(HistoricoRecuperado[valpag])
    43. _Recuperado_presunto

    Descripción: Para una empresa en particular si está en mora y en el mes siguiente no lo está, se toma como recuperado lo que debía en el mes anterior.

    Meta: Ver ficha técnica de indicadores de madurez

    
VAR perac = SELECTEDVALUE(Periodos[periodo])
VAR actYear = LEFT(perac, 4)
VAR actMonth = RIGHT(perac, 2)
VAR antPer = CONVERT(IF(actMonth = "01", (actYear - 1) * 100 + 12, (actYear) * 100 + (actMonth - 1)), STRING)
RETURN CALCULATE(SUM(HistoricoMoraEmp[valormor]), ALL(Periodos), Periodos[periodo] = antPer, HistoricoMoraEmp[Ultimo] = 1)
    44. %EmpNotificadas

    Descripción: Relación entre el total de empresas notificadas y las empresas en mora.

    
[_#EmpMoraNotificacion] / [_#EmpMora]
    45. _Metricas

SWITCH(
    SELECTEDVALUE(Metricas[Nombre]),
    "# Afiliaciones", SUM(Historico[seAfilio]),
    "# Retiros", SUM(Historico[seRetiro]),
    "# Activos", CALCULATE(DISTINCTCOUNT(Historico[cod]), Historico[validador] = 1),
    "Cobertura Proyección Empresas", [_IndicadorCumplimientoAfiliacionesEmp],
    ...
)
    46. _#Encuestas

    Descripción: Total de encuestas aplicadas.

    
DISTINCTCOUNTNOBLANK(Satisfaccion[Validador_total_encuestas])
    47. _%Polaridad

    Descripción: Participación de personas satisfechas e insatisfechas.

    
VAR periodo = SELECTEDVALUE(Periodos[periodo])
VAR polaridad = CALCULATE(
    COUNTROWS(Satisfaccion) /
    CALCULATE(COUNTROWS(Satisfaccion), REMOVEFILTERS(Satisfaccion[Polaridad])),
    ALL(Periodos),
    Satisfaccion[Periodo] = periodo
)
RETURN polaridad
    48. _Calificacion

    Descripción: Promedio de calificación de encuestas.

    
VAR periodo = SELECTEDVALUE(Periodos[periodo])
VAR avg_calif = CALCULATE(
    AVERAGE(Satisfaccion[Calificacion]),
    ALL(Periodos),
    Satisfaccion[Periodo] = periodo
)
RETURN avg_calif
    49. _cumplimientoAfiliacionesEncuestas

    Descripción: Relación entre el total de encuestas aplicadas y el muestreo sugerido.

    Meta: Dependiendo del total de la población con un 90% de confianza.

    
_cumplimientoAfiliacionesEncuestas = COUNT(Satisfaccion[Validador_total_encuestas]) / [_TamañoMuestra]
    50. _satisfaccionMesActualvsAnterior

    Descripción: Variación de la calificación por pregunta de encuesta, del periodo de estudio versus el periodo anterior.

    
VAR periodoActual = SELECTEDVALUE(Periodos[periodo])
VAR periodoAnterior = CALCULATE(
    MAX(Periodos[periodo]),
    ALL(Periodos),
    Periodos[periodo] < periodoActual
)
VAR encAnt = CALCULATE([_Calificacion], ALL(Periodos), Periodos[periodo] = periodoAnterior)
RETURN IF(ISBLANK(encAnt), BLANK(), ([_Calificacion] - encAnt) / encAnt)
    51. _satisfaccionMesActualvsAnterior(Año)

    52. _Metricas_tabla_afil

    Descripción: Consolida las cifras de afiliaciones activas mes anterior, afiliaciones en el periodo y desafiliaciones.

    
SWITCH(
    SELECTEDVALUE(TablaAfiliaciones[Criterio]),
    "Activos mes anterior", [_totalAfiliados] - SUM(Historico[seAfilio]),
    "Afiliaciones en el periodo", SUM(Historico[seAfilio]),
    "Desafiliaciones", -[_Desafiliaciones]
)
    53. _Metricas_tabla_apo

    Descripción: Consolida las cifras de aportes afiliados activos mes anterior, aportes no afiliados, aportes recuperados por mora presunta, aportes afiliados en el periodo y aportes recuperados por mora real.

    
SWITCH(
    SELECTEDVALUE(TablaAportes[Criterio]),
    "Aportes afiliados activos mes anterior", CALCULATE(SUM(HistoricoAportesEmp[valorpag]), Historico[seAfilio] <> 1),
    "Aportes afiliados en el periodo", CALCULATE(SUM(HistoricoAportesEmp[valorpag]), Historico[seAfilio] = 1),
    "Aportes no afiliados", [_$EmpNoAfiliadas],
    "Aportes recuperados por mora real", [_Recuperado],
    "Aportes recuperados por mora presunta", [_Recuperado_presunto]
